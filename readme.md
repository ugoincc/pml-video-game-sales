# 🎮 Video Games Sales Classifier
 
> Classificação de faixa de vendas de jogos eletrônicos com base em atributos pré-lançamento.
 
---
 
## Objetivo
 
Treinar um modelo de **classificação multiclasse** capaz de prever em qual faixa de vendas globais um jogo será enquadrado **antes de seu lançamento**, utilizando apenas informações disponíveis nesse momento: gênero, plataforma, publisher, desenvolvedor, nota da crítica e data de lançamento.
 
O problema tem aplicação direta em decisões de negócio da indústria de games — publishers e distribuidores precisam estimar o potencial comercial de um título para alocar orçamento de marketing, definir tiragem inicial e priorizar releases.
 
---
